### base包介绍
- 框架用到的基础库都放在这里
### base包枚举
- dailytimer  日常定时器库
- events      事件库
- file        文件库
- gameconfig  配置读取库
- linmath     数学库
- mongodbservcie   mongodb驱动操作库（基于开源mgo库封装）
- ranking      elo排行库
- safecontainer   安全可靠的队列实现库
- serializer    序列化、反序列化库
- stream        字节流的读、写库
- timer         定时器库
- utility       类型转换库
- zlog          日志库
